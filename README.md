# 🍯 العسّال — مساعد العسل الذكي

تطبيق Next.js مبني على نفس بنية "العطّار" لكن متخصص في عالم العسل الطبيعي.

## المميزات
- 🐝 مساعد ذكي متخصص بأنواع العسل وفوائده
- 🌿 تصنيف الأعسال: طبية، برية، زهرية
- 🛒 سلة طلبات
- 📖 دليل الجودة واختبارات الأصالة
- 🎨 ثيم ذهبي عسلي فاخر

## التشغيل

```bash
npm install
# أضف مفتاح API في .env.local
npm run dev
```

## متغيرات البيئة
```
ANTHROPIC_API_KEY=your_key_here
```
